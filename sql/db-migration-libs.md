# Database Migration Libraries

| Tool | Language | Notes |
| :--- | :--- | :--- |
| [Goose](https://github.com/pressly/goose) | Go | Native Go tool |
| [Flyway](https://flywaydb.org/) | Java, etc. | Simple file-based |
| [Liquibase](https://www.liquibase.com/) | Java | More config-heavy |
| [Alembic](https://alembic.sqlalchemy.org/) | Python | For SQLAlchemy |
| [Prisma Migrate](https://www.prisma.io/migrate) | Node.js | Works with Prisma ORM |
